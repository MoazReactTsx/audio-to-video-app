Notes
  • WebM is built first with VP9 or VP8, then converted locally to MP4 with ffmpeg.wasm.
  • Use 854 x 480 if the browser starts to struggle with large files.
  • Trim voice directly on the timeline by dragging the selection edges with the mouse.
  • File size is mainly driven by bitrate and duration.
